# movie-viewing-log-app

## アプリケーション概要

「」は映画好きな人たちのために作成した、個人向けの映画評価アプリです。
<!-- アプリの強み -->で、視聴した映画のジャンルと評価を元に、
自分の好みに合った映画を一目で分かるようにすることが出来ます。

<URL>

##　主な機能

・認証（ログイン画面）
・映画の登録 / 編集 / 削除（CRUD）
・「視聴済」「視聴希望」ステータス管理、評価（★1〜5）、感想の記録
・検索・フィルタ（ジャンル、評価、状態）
・RLSによるユーザーごとのデータ保護

##　使用技術

Frontend: Next.js（App Router）/ TypeScript / Tailwind CSS
Backend: Supabase（PostgreSQL / Auth / RLS）
Hosting: Vercel
